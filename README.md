# 🏢 ITCEN Group G2B Bid Crawler

## 프로젝트 소개 
조달청(나라장터) 공공데이터포털 API를 활용하여 아이티센 그룹(클로잇, 엔텍, 글로벌)의 주력 비즈니스 도메인에 맞는 IT/SI/클라우드 입찰 공고만 선별적으로 수집하는 자동화 파이프라인입니다.
노이즈 데이터를 제거하고, **'입찰공고문'과 '제안요청서(RFP)'가 모두 존재하는 공고만 엄선**하여 데이터 수집 및 수주 검토 업무의 효율성을 극대화합니다.

## 주요 기능 
* **계열사별 타겟팅 필터링:**
  * 조달청 8자리 세부분류명(물품분류코드)을 기반으로 하드 필터링 적용
  * `클로잇`: 클라우드, AI, 신규 시스템 구축, ISP 등
  * `엔텍`: 정보통신 인프라, 보안, 네트워크 구축 등
  * `글로벌`: 전산장비/SW 유지보수, ITO, 헬프데스크 등
* **지능형 핵심 문서 교차 검증 (고도화):**
  * 수많은 공고 중 **'제안요청서'와 '입찰공고' 첨부파일이 동시에 존재하는 공고만 최종 추출**하여, 실질적으로 제안 평가 및 입찰 참여가 가능한 대형 사업(협상에 의한 계약 등) 위주로 타겟팅합니다. 
  * 문서의 다이렉트 다운로드 링크(URL)를 함께 제공합니다.
* **데이터 파일 자동화:** 수집 및 필터링된 데이터를 가독성 높은 표 형태로 포맷팅하여 최종적으로 엑셀 파일로 한 번에 자동 저장 및 출력합니다.
* **실시간 진행 상황 모니터링:** 터미널 대시보드를 통해 API 호출 로딩 상태, 현재 진행 중인 데이터 페이지 및 건수, 최종 추출된 결과 갯수를 직관적으로 제공합니다.

## 기술 스택
* **Language:** Python 3.x
* **Library:** `requests`, `pandas`
* **API:** 공공데이터포털 조달청 용역입찰공고정보 API,  공공데이터포털 조달청 물품목록정보서비스 API

## 사전 준비 
본 스크립트를 실행하기 위해 다음 라이브러리 설치가 필요합니다.

```bash
pip install requests pandas
```
## 사용 방법

### 1. 조회 기간 및 수집 범위 설정
* 실행부에서 함수 파라미터를 조절하여 수집 기간 유연하게 설정 가능하빈다.

```python
# 예: 최근 7일(days_back=7) 데이터를 하루(chunk_days=1) 단위로 쪼개서 수집
results = get_itcen_group_bids(days_back=7, chunk_days=1)
```
### 2. 도메인 맞춤형 분류 코드 커스터마이징
  * 코드 내 `CLOIT_CODES`, `ENTEC_CODES`, `GLOBAL_CODES` 리스트에 회사별 도메인에 맞는 코드를 자유롭게 추가 및 수정하여 추출할 수 있습니다.
  * 함께 제공되는 `procurement_classification_table.csv` 파일(조달청 물품목록정보서비스 API를 통해 추출한 코드 목록)을 참조하여 성격에 맞는 신규 분류 코드를 발굴할 수 있습니다.
### 3. 실행 및 결과 확인
  * 실행 시 터미널 화면에 조회 중인 날짜, 현재 다운로드 중인 페이지 등의 실시간 진행률(로딩 상태)이 표시됩니다.
  * 탐색이 완료되면 필터링을 통과한 **최종 결과 개수**가 출력되며, 요약된 공고 정보가 대시보드에 나타난 후 **엑셀 파일로 일괄 저장**되어 활용 가능합니다.
