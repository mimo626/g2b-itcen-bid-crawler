# 🏢 ITCEN Group G2B Bid Crawler

## 프로젝트 소개 
조달청(나라장터) 공공데이터포털 API를 활용하여 **아이티센 그룹(클로잇, 엔텍, 글로벌)**의 주력 비즈니스 도메인에 맞는 IT/SI/클라우드 입찰 공고만 선별적으로 수집하는 자동화 파이프라인
노이즈 데이터를 제거하고, 입찰에 필수적인 **'제안요청서' 및 '과업지시서' 첨부파일 다운로드 링크를 즉시 추출**하여 영업 및 제안 팀의 업무 효율성을 극대화함

## 주요 기능 
* **계열사별 타겟팅 필터링:**
  * 조달청 8자리 세부분류명(물품분류코드)을 기반으로 하드 필터링 적용
  * `클로잇`: 클라우드, AI, 신규 시스템 구축, ISP 등
  * `엔텍`: 정보통신 인프라, 보안, 네트워크 구축 등
  * `글로벌`: 전산장비/SW 유지보수, ITO, 헬프데스크 등
* **첨부파일 자동 추출:** 수많은 첨부파일 중 '제안', '과업', '입찰' 키워드가 포함된 핵심 문서의 다이렉트 다운로드 링크(URL)만 선별 추출
* **데이터 파일 자동화:** 수집된 데이터를 가독성 높은 표 형태로 포맷팅하여 `.csv` (또는 `.xlsx`) 파일로 자동 저장

## 기술 스택
* **Language:** Python 3.x
* **Library:** `requests`, `pandas`
* **API:** 공공데이터포털 조달청 용역입찰공고정보 API,  공공데이터포털 조달청 물품목록정보서비스 API

## 사전 준비 
본 스크립트를 실행하기 위해 다음 라이브러리 설치 필요

```bash
pip install requests pandas
```
## 사용 방법
*  조회 기간 설정을 바꿔가며 새로운 데이터 갱신 가능함
*  변수 result_count로 결과 값의 개수를 변경할 수 있음(단, 필터링되어 개수가 줄어듦)
*  물품목록정보서비스의 분류 코드를 토대로 각 회사의 그룹사에 맞는 카테고리 코드로 설정해 뒀으며 추가할 수 있음
